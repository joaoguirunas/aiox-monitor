# Story 1.1 — Setup do Projeto

## Story

**Como** desenvolvedor do aiox-monitor,
**Quero** ter o projeto Next.js configurado com TypeScript, Tailwind e todas as dependências necessárias,
**Para que** as stories seguintes possam ser implementadas sem blockers de infraestrutura.

## Dev Notes

### Stack Técnico
- **Next.js** App Router (não Pages Router)
- **Porta:** 8888 — configurar via `next dev --port 8888` no script `dev`
- **better-sqlite3** é um módulo nativo Node.js — OBRIGATÓRIO `serverExternalPackages` no next.config para evitar erro de bundling
- **Phaser.js** só roda no client-side — usar `dynamic(() => import(...), { ssr: false })` quando integrado
