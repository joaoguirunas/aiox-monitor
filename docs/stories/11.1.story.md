# Story 11.1 — ProcessManager Backend

## Story

**Como** desenvolvedor do AIOX Monitor,
**Quero** um ProcessManager singleton que gerencie processos Claude Code via node-pty,
**Para que** o backend possa spawnar, matar, redimensionar e listar pseudo-terminais de agentes.

## Dev Notes

- `node-pty` requer build tools nativos (Xcode CLT no macOS, build-essential no Linux)
- Usar `process.platform` para detectar shell: `/bin/zsh -l` (macOS), `/bin/bash -l` (Linux), `powershell.exe` (Windows)
- O comando spawned é: `claude --agent {agentName}` dentro do `projectPath`
- UUID v4 para IDs de processo (usar `crypto.randomUUID()`)
- Processos fechados ficam em memória 30s para reconexão, depois são removidos
- `node-pty` deve ser adicionado a `serverExternalPackages` no `next.config.ts`
