# 11.4 Frontend Terminal Panel — xterm.js + Aba Sala de Comando

## Objetivo
Criar o componente frontend que renderiza terminais reais no browser usando xterm.js, conectados ao backend via WebSocket. Renomear a aba Terminais para Sala de Comando.

## Acceptance Criteria
- [ ] AC1: xterm.js e addons instalados (xterm, @xterm/addon-fit, @xterm/addon-web-links)
- [ ] AC2: Componente TerminalPanel.tsx renderiza terminal funcional
- [ ] AC3: Hook usePtySocket.ts conecta ao WebSocket /pty
- [ ] AC4: Aba 'Terminais' renomeada para 'Sala de Comando' no Navbar
- [ ] AC5: Página /sala-de-comando com botão 'Iniciar Terminal'
- [ ] AC6: Terminal mostra output real de processo Claude spawned
- [ ] AC7: Input do usuário é enviado para o processo via WebSocket
- [ ] AC8: Terminal pode ser fechado (mata o processo)
- [ ] AC9: Theme dark AIOX aplicado ao xterm.js
- [ ] AC10: Cross-platform funcional (Mac/Linux/Win)
