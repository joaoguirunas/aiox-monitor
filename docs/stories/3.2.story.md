# Story 3.2 — Tilemap do Escritório Isométrico

## Story

**Como** aiox-monitor,
**Quero** renderizar um escritório isométrico com tilemap dividido em 3 zonas (trabalho, lounge, entrada),
**Para que** o Modo Empresa tenha um espaço visual organizado onde os agentes vão se posicionar.

## Acceptance Criteria

- [ ] **AC1** — `src/game/scenes/OfficeScene.ts` renderiza um tilemap isométrico usando tiles programáticos (sem assets externos na v1)
- [ ] **AC2** — O mapa tem 3 zonas visuais distintas:
  - **Zona Trabalho** (centro-direita): área de mesas com grid para até 11 posições de agente
  - **Zona Lounge** (centro-esquerda): sofás e mesa de café para agentes `idle`/`break`

- [ ] **AC4** — Paredes e limites do escritório são desenhados nas bordas do mapa
- [ ] **AC5** — A câmara Phaser está posicionada para ver o escritório inteiro por default
- [ ] **AC6** — Zoom in/out com scroll do mouse (limites: 0.5x a 2x)

- [ ] **AC8** — `OfficeScene` substitui `BootScene` como cena principal após carregamento
- [ ] **AC9** — As coordenadas (tile) de cada posição de agente estão definidas em `src/game/data/office-layout.ts`

- [x] **Task 2** — Criar `src/game/utils/iso-utils.ts` — conversão de coordenadas (AC: 1)
  ```typescript
  import { TILE_WIDTH, TILE_HEIGHT } from '../data/office-layout';

  // Converte coordenadas tile (grid) para pixel (screen) em projeção isométrica
  export function tileToPixel(tileX: number, tileY: number): { x: number; y: number } {
    return {
      x: (tileX - tileY) * (TILE_WIDTH / 2),
      y: (tileX + tileY) * (TILE_HEIGHT / 2),
    };
  }

  // Converte coordenadas pixel para tile (para clicks/interação)
  export function pixelToTile(pixelX: number, pixelY: number): { tileX: number; tileY: number } {
    return {
      tileX: Math.floor((pixelX / (TILE_WIDTH / 2) + pixelY / (TILE_HEIGHT / 2)) / 2),
      tileY: Math.floor((pixelY / (TILE_HEIGHT / 2) - pixelX / (TILE_WIDTH / 2)) / 2),
    };
  }
  ```

  import Phaser from 'phaser';
  import {
    MAP_WIDTH, MAP_HEIGHT, TILE_WIDTH, TILE_HEIGHT,
    getZoneForTile, ZONES
  } from '../data/office-layout';
  import { tileToPixel } from '../utils/iso-utils';

  export class OfficeScene extends Phaser.Scene {
    constructor() {
      super({ key: 'OfficeScene' });
    }

    create(): void {
      this.drawFloor();
      this.drawWalls();
      this.setupCamera();
    }

    private drawFloor(): void {
      const graphics = this.add.graphics();

      for (let y = 0; y < MAP_HEIGHT; y++) {
        for (let x = 0; x < MAP_WIDTH; x++) {
          const zone = getZoneForTile(x, y);
          if (!zone) continue;

          const zoneColor = ZONES.find(z => z.name === zone)?.color ?? 0x222233;
          const { x: px, y: py } = tileToPixel(x, y);

          // Desenha losango isométrico
          graphics.fillStyle(zoneColor, 1);
          graphics.beginPath();
          graphics.moveTo(px, py - TILE_HEIGHT / 2);           // top
          graphics.lineTo(px + TILE_WIDTH / 2, py);             // right
          graphics.lineTo(px, py + TILE_HEIGHT / 2);            // bottom
          graphics.lineTo(px - TILE_WIDTH / 2, py);             // left
          graphics.closePath();
          graphics.fillPath();

          // Borda sutil do tile
          graphics.lineStyle(1, 0x333344, 0.3);
          graphics.strokePath();
        }
      }
    }

    private drawWalls(): void {
      // Implementar paredes como linhas nas bordas das zonas
      const graphics = this.add.graphics();
      graphics.lineStyle(2, 0x555577, 0.8);
      // Borda exterior do escritório — conectar tiles de borda
      // Simplificado: retângulo isométrico ao redor do mapa
    }

    private setupCamera(): void {
      // Centralizar no meio do mapa
      const center = tileToPixel(MAP_WIDTH / 2, MAP_HEIGHT / 2);
      this.cameras.main.centerOn(center.x, center.y);

      // Zoom com scroll do mouse
      this.input.on('wheel', (
        _pointer: Phaser.Input.Pointer,
        _gameObjects: Phaser.GameObjects.GameObject[],
        _deltaX: number,
        deltaY: number
      ) => {
        const zoom = this.cameras.main.zoom;
        const newZoom = Phaser.Math.Clamp(zoom - deltaY * 0.001, 0.5, 2);
        this.cameras.main.setZoom(newZoom);
      });

      // Pan com drag
      this.input.on('pointermove', (pointer: Phaser.Input.Pointer) => {
        if (!pointer.isDown) return;
        this.cameras.main.scrollX -= (pointer.x - pointer.prevPosition.x) / this.cameras.main.zoom;
        this.cameras.main.scrollY -= (pointer.y - pointer.prevPosition.y) / this.cameras.main.zoom;
      });
    }
  }

## Dev Notes

### Tiles programáticos vs assets

Na v1, os tiles são desenhados com `Graphics` do Phaser (losangos coloridos). Não há spritesheet de tilemap nesta fase — será adicionado em stories futuras quando o visual evoluir. Isto permite iterar rápido sem depender de pixel art.

### Projeção isométrica — fórmula

A conversão tile→pixel usa a fórmula padrão isométrica 2:1:
```
pixelX = (tileX - tileY) * (TILE_WIDTH / 2)
pixelY = (tileX + tileY) * (TILE_HEIGHT / 2)
```

Com `TILE_WIDTH=64` e `TILE_HEIGHT=32`, cada tile é um losango de 64x32 pixels.

### Layout das zonas

```
+------------------+
|   [TRABALHO]     |  x=10-18, y=1-13
|   11 mesas       |  Grid 3x4 (última fila tem 2)
|                  |
| [LOUNGE]         |  x=1-7, y=3-11
| sofás + café     |
|                  |
|    [ENTRADA]     |  x=7-12, y=12-15
|    porta         |
+------------------+
```

### Camera bounds

O zoom é limitado a 0.5x-2x para evitar perder o escritório de vista. O pan é livre — o utilizador pode arrastar para qualquer lado.

### Performance

Tiles desenhados com `Graphics` uma vez no `create()` — não há re-draw por frame. O custo é fixo: 20x16 = 320 losangos. Performante em qualquer hardware.
