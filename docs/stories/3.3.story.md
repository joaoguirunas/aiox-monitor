- [x] **Task 1** — Criar `src/game/objects/Desk.ts` (AC: 1, 2, 9)
  ```typescript
  import Phaser from 'phaser';
  import { tileToPixel } from '../utils/iso-utils';

  export class Desk extends Phaser.GameObjects.Container {
    private screen: Phaser.GameObjects.Graphics;
    private occupied = false;

    constructor(scene: Phaser.Scene, tileX: number, tileY: number) {
      const { x, y } = tileToPixel(tileX, tileY);
      super(scene, x, y);

      // Mesa: retângulo isométrico achatado
      const table = scene.add.graphics();
      table.fillStyle(0x4a3728, 1); // madeira escura
      table.fillRect(-20, -6, 40, 12);
      this.add(table);

      // Monitor: retângulo vertical
      const monitor = scene.add.graphics();
      monitor.fillStyle(0x333333, 1);
      monitor.fillRect(-8, -20, 16, 14);
      this.add(monitor);

      // Tela do monitor
      this.screen = scene.add.graphics();
      this.setScreenOff();
      this.add(this.screen);

      this.setDepth(y); // depth sorting
      scene.add.existing(this);
    }

    setScreenOn(color: number = 0x44ff44): void {
      this.screen.clear();
      this.screen.fillStyle(color, 0.8);
      this.screen.fillRect(-6, -18, 12, 10);
      this.occupied = true;
    }

    setScreenOff(): void {
      this.screen.clear();
      this.screen.fillStyle(0x111111, 1);
      this.screen.fillRect(-6, -18, 12, 10);
      this.occupied = false;
    }

    isOccupied(): boolean {
      return this.occupied;
    }
  }
  ```

## Dev Notes

### Sprites programáticos — v1

Todos os móveis são desenhados com `Graphics` do Phaser (retângulos, círculos, linhas). Isto é intencional para a v1 — permite iterar rápido. Spritesheets pixel art serão adicionados em stories futuras (fora do escopo da Fase 3 básica).

### Depth sorting

O `setDepth(y)` usa a coordenada Y em pixels como profundidade. Objetos com Y maior (mais ao sul na projeção isométrica) aparecem na frente. Isto garante o efeito de profundidade correto.

### Container pattern

Cada móvel é um `Phaser.GameObjects.Container` que agrupa seus sub-elementos gráficos. Isto permite mover, escalar ou destruir o móvel como unidade.

### Desk screen state

`Desk.setScreenOn(color)` / `setScreenOff()` será usado na Story 3.6 quando o agente se sentar. Nesta story, todas as mesas começam com tela apagada.

### Cores dos agentes para telas

As cores de `agent-colors.ts` (hex) serão mapeadas para cores Phaser (number hex) na Story 3.6. Exemplo: `bg-blue-500` → `0x3b82f6`.
