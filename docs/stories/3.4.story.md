- [x] **Task 2** — Criar `src/game/objects/AgentSprite.ts` (AC: 1, 2, 3, 4, 5, 6, 8)
  ```typescript
  import Phaser from 'phaser';
  import { tileToPixel } from '../utils/iso-utils';
  import { getAgentColor, STATUS_COLORS } from '../data/agent-visuals';
  import type { AgentStatus } from '@/lib/types';

  export class AgentSprite extends Phaser.GameObjects.Container {
    private bodyGraphics: Phaser.GameObjects.Graphics;
    private statusBadge: Phaser.GameObjects.Graphics;
    private nameLabel: Phaser.GameObjects.Text;
    private agentColor: number;
    private currentStatus: AgentStatus = 'idle';

    constructor(
      scene: Phaser.Scene,
      public readonly agentName: string,
      public readonly displayName: string,
      tileX: number,
      tileY: number,
    ) {
      const { x, y } = tileToPixel(tileX, tileY);
      super(scene, x, y);

      this.agentColor = getAgentColor(agentName);

      // Corpo: elipse (perspectiva isométrica)
      this.bodyGraphics = scene.add.graphics();
      this.drawBody();
      this.add(this.bodyGraphics);

      // Status badge
      this.statusBadge = scene.add.graphics();
      this.drawStatusBadge();
      this.add(this.statusBadge);

      // Nome flutuante
      this.nameLabel = scene.add.text(0, -32, displayName, {
        fontSize: '10px',
        color: '#ffffff',
        fontFamily: 'monospace',
        backgroundColor: '#00000088',
        padding: { x: 3, y: 1 },
      }).setOrigin(0.5);
      this.add(this.nameLabel);

      this.setDepth(y + 1); // acima dos móveis na mesma posição Y
      scene.add.existing(this);
    }

    private drawBody(): void {
      this.bodyGraphics.clear();
      // Cabeça (círculo)
      this.bodyGraphics.fillStyle(0xddccbb, 1); // tom de pele
      this.bodyGraphics.fillCircle(0, -18, 6);
      // Corpo (elipse/retângulo arredondado com cor do agente)
      this.bodyGraphics.fillStyle(this.agentColor, 1);
      this.bodyGraphics.fillRoundedRect(-7, -12, 14, 16, 3);
      // Pernas (dois retângulos finos)
      this.bodyGraphics.fillStyle(0x333344, 1);
      this.bodyGraphics.fillRect(-5, 4, 4, 6);
      this.bodyGraphics.fillRect(1, 4, 4, 6);
    }

    private drawStatusBadge(): void {
      this.statusBadge.clear();
      const color = STATUS_COLORS[this.currentStatus] ?? STATUS_COLORS.idle;
      this.statusBadge.fillStyle(color, 1);
      this.statusBadge.fillCircle(10, -18, 4);
      // Borda branca
      this.statusBadge.lineStyle(1, 0xffffff, 0.8);
      this.statusBadge.strokeCircle(10, -18, 4);
    }

    setStatus(status: AgentStatus): void {
      this.currentStatus = status;
      this.drawStatusBadge();
    }

    setTilePosition(tileX: number, tileY: number): void {
      const { x, y } = tileToPixel(tileX, tileY);
      this.setPosition(x, y);
      this.setDepth(y + 1);
    }

    getAgentName(): string {
      return this.agentName;
    }
  }
  ```

## Dev Notes

### Personagem programático — v1

O personagem é uma composição de formas geométricas (círculo=cabeça, retângulo=corpo, retângulos=pernas). Estilo "chibi isométrico simplificado". Em stories futuras, pode ser substituído por spritesheet pixel art, mas o Container pattern mantém a API igual.

### Depth sorting — agentes vs móveis

Agentes usam `setDepth(y + 1)` — o `+1` garante que aparecem acima dos móveis na mesma coordenada Y. Móveis usam `setDepth(y)`. Agentes mais ao sul continuam a aparecer na frente de agentes mais ao norte.

### Cor por agente

Cada agente tem cor primária derivada das cores Tailwind já definidas em `agent-colors.ts`. O mapeamento em `agent-visuals.ts` converte nomes de cor Tailwind para hex Phaser.

### Container pattern

`AgentSprite extends Phaser.GameObjects.Container` agrupa corpo + badge + nome. `setTilePosition()` move tudo junto. `destroy()` limpa tudo. Consistente com os objetos de mobiliário.

### Agente de teste — temporário

O `placeTestAgent()` é uma prova de conceito. Na Story 3.6, os agentes serão criados dinamicamente a partir dos dados reais do banco (via React→Phaser bridge).
