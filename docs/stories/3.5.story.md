# Story 3.5 — Animações Básicas

## Story

**Como** aiox-monitor,
**Quero** que os personagens dos agentes tenham animações de caminhar, sentar, digitar e idle,
**Para que** o escritório tenha vida e o estado de cada agente seja visualmente intuitivo.

## Acceptance Criteria

- [x] **AC1** — `AgentSprite` suporta 4 estados visuais: `walk`, `sit`, `type`, `idle`
- [x] **AC2** — **Walk**: personagem se move suavemente entre tiles usando tween (não teleporta)
- [x] **AC3** — **Sit**: corpo do personagem abaixa ligeiramente e as pernas ficam ocultas (sentado na cadeira)
- [x] **AC4** — **Type**: quando sentado, mãos do personagem "vibram" sutilmente (tween yoyo em x) simulando digitação
- [x] **AC5** — **Idle**: personagem fica em pé, com balanço sutil de respiração (tween yoyo em y, ±1px, lento)
- [x] **AC6** — `walkTo(tileX, tileY)` retorna uma Promise que resolve quando o agente chega ao destino
- [x] **AC7** — Durante o walk, o personagem segue um caminho em linha reta (pathfinding simples)
- [x] **AC8** — O depth sorting é atualizado durante o walk (a cada frame do tween)
- [x] **AC9** — Transições entre estados são suaves (ex: type→idle para por cima e depois levanta)
- [x] **AC10** — `npm run typecheck` e `npm run build` passam sem erros

- [x] **Task 1** — Criar `src/game/animations/agent-animations.ts` (AC: 1, 3, 4, 5)
  ```typescript
  import Phaser from 'phaser';

  export type AgentAnimState = 'walk' | 'sit' | 'type' | 'idle';

  // Aplica animação de idle (respiração sutil)
  export function playIdle(
    scene: Phaser.Scene,
    container: Phaser.GameObjects.Container,
  ): Phaser.Tweens.Tween {
    return scene.tweens.add({
      targets: container,
      y: container.y - 1,
      duration: 2000,
      yoyo: true,
      repeat: -1,
      ease: 'Sine.easeInOut',
    });
  }

  // Aplica animação de sit (abaixar corpo)
  export function playSit(
    scene: Phaser.Scene,
    bodyGraphics: Phaser.GameObjects.Graphics,
  ): Phaser.Tweens.Tween {
    return scene.tweens.add({
      targets: bodyGraphics,
      y: bodyGraphics.y + 4,
      duration: 300,
      ease: 'Power2',
    });
  }

  // Aplica animação de type (vibração das mãos)
  export function playType(
    scene: Phaser.Scene,
    bodyGraphics: Phaser.GameObjects.Graphics,
  ): Phaser.Tweens.Tween {
    return scene.tweens.add({
      targets: bodyGraphics,
      x: bodyGraphics.x + 1,
      duration: 100,
      yoyo: true,
      repeat: -1,
      ease: 'Linear',
    });
  }
  ```

- [x] **Task 2** — Atualizar `AgentSprite.ts` — estados e walkTo (AC: 1, 2, 6, 7, 8)
  ```typescript
  import {
    AgentAnimState, playIdle, playSit, playType
  } from '../animations/agent-animations';
  import { tileToPixel } from '../utils/iso-utils';

  // Novos campos no AgentSprite:
  private animState: AgentAnimState = 'idle';
  private currentTween: Phaser.Tweens.Tween | null = null;

  // Walk para uma posição tile com tween
  walkTo(tileX: number, tileY: number): Promise<void> {
    return new Promise((resolve) => {
      this.stopCurrentAnimation();
      this.animState = 'walk';

      const { x: targetX, y: targetY } = tileToPixel(tileX, tileY);
      const distance = Phaser.Math.Distance.Between(this.x, this.y, targetX, targetY);
      const duration = Math.max(distance * 4, 500); // 4ms por pixel, mínimo 500ms

      this.currentTween = this.scene.tweens.add({
        targets: this,
        x: targetX,
        y: targetY,
        duration,
        ease: 'Power1',
        onUpdate: () => {
          // Atualizar depth sorting durante walk
          this.setDepth(this.y + 1);
        },
        onComplete: () => {
          this.animState = 'idle';
          resolve();
        },
      });
    });
  }

  // Transicionar para estado sit
  sitDown(): void {
    this.stopCurrentAnimation();
    this.animState = 'sit';
    this.currentTween = playSit(this.scene, this.bodyGraphics);
  }

  // Transicionar para estado type (deve estar sentado)
  startTyping(): void {
    this.animState = 'type';
    this.currentTween = playType(this.scene, this.bodyGraphics);
  }

  // Transicionar para idle (parado em pé)
  standIdle(): void {
    this.stopCurrentAnimation();
    this.animState = 'idle';
    // Reset body position
    this.bodyGraphics.setPosition(0, 0);
    this.currentTween = playIdle(this.scene, this);
  }

  private stopCurrentAnimation(): void {
    if (this.currentTween) {
      this.currentTween.stop();
      this.currentTween = null;
    }
  }

  // Override destroy para limpar tweens
  destroy(fromScene?: boolean): void {
    this.stopCurrentAnimation();
    super.destroy(fromScene);
  }
  ```

## Dev Notes

### Tween-based animations

Todas as animações usam o sistema de `Tweens` do Phaser — não são sprite animations baseadas em frames. Isto é ideal para a v1 com personagens programáticos (Graphics). Se no futuro migrarmos para spritesheets, os tweens de posição (walk) continuam, e as animações visuais (sit, type, idle) são substituídas por `anims.play()`.

### Walk path — linha reta

Na v1, o agente caminha em linha reta entre dois pontos. Não há pathfinding com obstáculos. Se precisarmos de pathfinding no futuro, implementar A* ou usar plugin `easystar.js` — fora do escopo desta story.

### walkTo como Promise

`walkTo()` retorna Promise — permite sequenciar ações com `async/await`:
```typescript
await agent.walkTo(desk.tileX, desk.tileY);
agent.sitDown();
```

Isto simplifica enormemente a lógica de comportamento na Story 3.7.

### Depth sorting durante walk

O `onUpdate` callback do tween atualiza `setDepth(this.y + 1)` a cada frame. Sem isto, o agente ficaria "por trás" de móveis que deveria passar "na frente" durante o percurso.

### Cleanup de tweens

O `destroy()` chama `stopCurrentAnimation()` que para o tween activo. Sem isto, o tween continua a executar em objectos destruídos — causa erros no console.

### Duração do walk

`distance * 4` = 4ms por pixel de distância. Para um tile (64x32), a diagonal é ~72px = ~288ms. Mínimo de 500ms para walks curtos não parecerem instantâneos.
