# Story 4.3 — Break Room Expandida

## Story

**Como** aiox-monitor,
**Quero** que a zona lounge seja expandida para uma break room completa com máquina de café, estante, planta, dispensador de água e posições de interacção por móvel,
**Para que** os agentes em break/idle tenham locais específicos para ir e o escritório pareça um espaço real de trabalho.

## Acceptance Criteria

- [ ] **AC1** — Novos objectos de mobiliário: `CoffeeMachine`, `Bookshelf`, `Plant`, `WaterCooler` em `src/game/objects/`
- [ ] **AC2** — Cada objecto estende `Phaser.GameObjects.Container` e é desenhado com Graphics programático (consistente com Desk/Sofa)
- [ ] **AC3** — `office-layout.ts` define posições dos novos móveis na zona lounge expandida
- [ ] **AC4** — A zona lounge é expandida de 4 para 8+ posições de interacção, incluindo posições "em pé perto de" cada móvel
- [ ] **AC5** — Agentes com status `break` vão para posições de interacção específicas: café, água ou estante (rotação round-robin)
- [ ] **AC6** — Agentes com status `idle` vão para sofás ou ficam em pé na zona lounge
- [ ] **AC7** — A `CoffeeMachine` tem animação: quando um agente está perto, emite "vapor" (partículas simples ou tween)
- [ ] **AC8** — O `AgentManager.getLoungePosition()` é refatorado para usar o mapa expandido de posições
- [ ] **AC9** — `npm run typecheck` e `npm run build` passam sem erros
- [ ] **AC10** — Teste visual: 3+ agentes idle/break distribuem-se pelos novos móveis sem sobreposição

  // src/game/objects/CoffeeMachine.ts
  export class CoffeeMachine extends Phaser.GameObjects.Container {
    private steamEmitter: Phaser.GameObjects.Particles.ParticleEmitter | null = null;

    constructor(scene: Phaser.Scene, tileX: number, tileY: number) {
      const { x, y } = tileToPixel(tileX, tileY);
      super(scene, x, y);

      const g = scene.add.graphics();
      // Base (caixa metálica)
      g.fillStyle(0x666677, 1);
      g.fillRoundedRect(-8, -16, 16, 16, 2);
      // Bico
      g.fillStyle(0x444455, 1);
      g.fillRect(-2, -18, 4, 3);
      // Chávena
      g.fillStyle(0xffffff, 1);
      g.fillRect(-3, -3, 6, 4);
      // LED indicador
      g.fillStyle(0x22c55e, 1);
      g.fillCircle(5, -14, 2);
      this.add(g);

      this.setDepth(y);
      scene.add.existing(this);
    }

    setSteamActive(active: boolean): void {
      // Partículas simples de vapor quando agente está perto
      if (active && !this.steamEmitter) {
        // Usar tween em vez de particle system para simplicidade
        const steam = this.scene.add.graphics();
        steam.fillStyle(0xffffff, 0.3);
        steam.fillCircle(0, -22, 3);
        this.add(steam);
        this.scene.tweens.add({
          targets: steam,
          y: steam.y - 8,
          alpha: 0,
          duration: 1000,
          repeat: -1,
          ease: 'Sine.easeOut',
        });
        this.steamEmitter = steam as unknown as Phaser.GameObjects.Particles.ParticleEmitter;
      } else if (!active && this.steamEmitter) {
        (this.steamEmitter as unknown as Phaser.GameObjects.Graphics).destroy();
        this.steamEmitter = null;
      }
    }
  }

  ```typescript
  import { LOUNGE_POSITIONS_V2, type LoungePosition } from '../data/office-layout';

  private getLoungePosition(name: string, status: AgentStatus): TilePosition {
    // Filtrar posições disponíveis para este status
    const available = LOUNGE_POSITIONS_V2.filter(
      (p) => p.interactionFor.includes(status as 'idle' | 'break')
    );

    // Encontrar posições não ocupadas
    const occupiedPositions = new Set<string>();
    for (const [n, sprite] of this.sprites) {
      if (n !== name) {
        occupiedPositions.add(`${Math.round(sprite.x)},${Math.round(sprite.y)}`);
      }
    }

    const free = available.filter((p) => {
      const { x, y } = tileToPixel(p.tileX, p.tileY);
      return !occupiedPositions.has(`${Math.round(x)},${Math.round(y)}`);
    });

    // Se há posição livre, usar. Senão, round-robin nas disponíveis
    if (free.length > 0) {
      const idx = [...this.sprites.keys()].indexOf(name) % free.length;
      return free[idx];
    }
    const idx = [...this.sprites.keys()].indexOf(name) % available.length;
    return available[idx];
  }
  ```

### Layout expandido do lounge

```

  [Bookshelf] (2,4)     [CoffeeMachine] (5,4)    [Plant] (7,3)
  [Sofá 1] (3,5)        [Standing] (4,6)          [CoffeeTable] (5,6)

                         [WaterCooler] (6,8)
  [Sofá 3] (3,9)        [Standing] (5,10)
```

### Lógica de posicionamento

- **idle** → sofás primeiro, depois standing overflow, depois bookshelf
- **break** → coffee-machine, water-cooler, bookshelf (round-robin)
- Posições não se sobrepõem: verifica posições ocupadas antes de atribuir
- Se não há posições livres, faz round-robin nas disponíveis (agentes podem partilhar zona)

### Interacção CoffeeMachine

A máquina de café tem um efeito visual simples: "vapor" (círculo branco semi-transparente a subir e desaparecer). É activado quando um agente em break está na posição `coffee-machine` e desactivado quando ninguém está perto.

### Backward compatibility

`LOUNGE_POSITIONS` original mantém-se exportado para não quebrar referências existentes. `LOUNGE_POSITIONS_V2` adiciona o `type` e `interactionFor`. O `AgentManager` usa V2 internamente.
