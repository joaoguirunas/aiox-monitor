# Story 4.4 — Sistema de Temas Visuais

## Story

**Como** aiox-monitor,
**Quero** que o escritório virtual suporte 4 temas visuais (espacial, moderno, oldschool, cyberpunk) que alteram cores do chão, paredes, móveis e efeitos ambientais,
**Para que** cada empresa possa personalizar a aparência do seu escritório virtual.

## Acceptance Criteria

- [ ] **AC1** — `src/game/data/themes.ts` define os 4 temas com paletas de cores completas: chão, paredes, móveis, efeitos
- [ ] **AC2** — Cada tema define: `floorColors` (por zona), `wallColor`, `wallAlpha`, `furnitureBaseColor`, `screenGlowColor`, `ambientParticles`, `backgroundColor`
- [ ] **AC3** — `OfficeScene` aplica o tema activo ao renderizar chão, paredes e fundo
- [ ] **AC4** — Os objectos de mobiliário (Desk, Sofa, CoffeeTable, etc.) recebem cores do tema via método `applyTheme(theme)`
- [ ] **AC5** — O tema é lido da `company_config` via API no arranque (`GET /api/company-config`)
- [ ] **AC6** — Tema pode mudar em runtime via WS message `theme:change` — a cena re-renderiza sem reiniciar
- [ ] **AC7** — Efeitos ambientais por tema: espacial (estrelas flutuantes), cyberpunk (scanlines), moderno (clean/sem efeitos), oldschool (grain/noise sutil)
- [ ] **AC8** — API endpoint `GET /api/company-config` retorna a config da empresa incluindo tema
- [ ] **AC9** — `npm run typecheck` e `npm run build` passam sem erros
- [ ] **AC10** — Teste visual: mudar tema via DB → recarregar → escritório com novo tema aplicado

- [x] **Task 4** — Refatorar `OfficeScene` para usar tema (AC: 3, 7)
  ```typescript
  import { getTheme, type OfficeTheme } from '../data/themes';
  import type { ThemeName } from '@/lib/types';

  export class OfficeScene extends Phaser.Scene {
    private currentTheme!: OfficeTheme;
    private floorGraphics!: Phaser.GameObjects.Graphics;
    private wallGraphics!: Phaser.GameObjects.Graphics;
    private ambientLayer: Phaser.GameObjects.Graphics | null = null;

    setTheme(themeName: ThemeName): void {
      this.currentTheme = getTheme(themeName);
      this.cameras.main.setBackgroundColor(this.currentTheme.backgroundColor);
      this.redrawFloor();
      this.redrawWalls();
      this.applyAmbientEffect();
      // Notificar mobiliário
      this.desks.forEach((d) => d.applyTheme(this.currentTheme));
    }

    private redrawFloor(): void {
      this.floorGraphics.clear();
      // Mesmo código de drawFloor, mas com cores do tema
      const hw = TILE_WIDTH / 2;
      const hh = TILE_HEIGHT / 2;
      for (let y = 0; y < MAP_HEIGHT; y++) {
        for (let x = 0; x < MAP_WIDTH; x++) {
          const zone = getZoneForTile(x, y);
          if (!zone) continue;
          const color = this.currentTheme.floorColors[zone];
          const { x: px, y: py } = tileToPixel(x, y);
          this.floorGraphics.fillStyle(color, 1);
          // ... diamond path
          this.floorGraphics.lineStyle(1, this.currentTheme.floorGridColor, this.currentTheme.floorGridAlpha);
          this.floorGraphics.strokePath();
        }
      }
    }

    private applyAmbientEffect(): void {
      if (this.ambientLayer) { this.ambientLayer.destroy(); this.ambientLayer = null; }

      const t = this.currentTheme;
      if (t.ambientEffect === 'none') return;

      if (t.ambientEffect === 'stars') {
        // Pontos brancos cintilantes no fundo
        const g = this.add.graphics().setDepth(-1);
        for (let i = 0; i < 50; i++) {
          g.fillStyle(t.ambientColor, Math.random() * t.ambientAlpha + 0.1);
          g.fillCircle(
            Phaser.Math.Between(-200, 1200),
            Phaser.Math.Between(-200, 600),
            Phaser.Math.Between(1, 2),
          );
        }
        this.ambientLayer = g;
        // Tween para cintilar
        this.tweens.add({
          targets: g, alpha: 0.3, duration: 3000,
          yoyo: true, repeat: -1, ease: 'Sine.easeInOut',
        });
      }

      if (t.ambientEffect === 'scanlines') {
        const g = this.add.graphics().setDepth(9999);
        g.fillStyle(0x000000, t.ambientAlpha);
        for (let y = 0; y < 800; y += 4) {
          g.fillRect(-400, y, 2000, 2);
        }
        this.ambientLayer = g;
      }

      if (t.ambientEffect === 'grain') {
        // Noise overlay sutil
        const g = this.add.graphics().setDepth(9999);
        g.fillStyle(t.ambientColor, t.ambientAlpha);
        for (let i = 0; i < 200; i++) {
          g.fillRect(
            Phaser.Math.Between(-200, 1200),
            Phaser.Math.Between(-200, 600),
            1, 1,
          );
        }
        this.ambientLayer = g;
      }
    }
  }
  ```

## Dev Notes

### Paleta de cores por tema

| Tema | Chão | Paredes | Móveis | Telas | Ambiente |
|------|------|---------|--------|-------|----------|
| Moderno | Cinza azulado | Cinza claro | Cinza escuro | Azul | Limpo |
| Espacial | Azul escuro | Azul médio | Azul metálico | Cyan | Estrelas |
| Oldschool | Marrom quente | Bege | Madeira | Verde CRT | Grain |
| Cyberpunk | Roxo escuro | Rosa neon | Preto | Magenta | Scanlines |

### Re-render sem reiniciar

O `setTheme()` limpa os Graphics existentes (`floorGraphics.clear()`, `wallGraphics.clear()`) e redesenha com as novas cores. Não reinicia a cena — sprites de agentes e suas posições são preservados.

### Efeitos ambientais

- **Estrelas (espacial)**: 50 pontos brancos com alpha variável, tween de cintilação
- **Scanlines (cyberpunk)**: linhas horizontais pretas a cada 4px, alpha muito baixo (0.03)
- **Grain (oldschool)**: 200 pixels aleatórios em cor quente, alpha 0.05
- **Nenhum (moderno)**: clean, sem overlay

### Backward compatibility

O tema default é `'moderno'`, que usa as mesmas cores que a Fase 3. Se a `company_config` não existir ou tema for inválido, usa `'moderno'` como fallback.
