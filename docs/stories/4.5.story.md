# Story 4.5 — Personalização da Empresa

## Story

**Como** utilizador do aiox-monitor,
**Quero** uma página de configuração onde possa definir o nome da empresa, selecionar o tema visual, e ajustar timeouts de idle,
**Para que** possa personalizar o escritório virtual e adaptar o comportamento dos agentes às minhas preferências.

## Acceptance Criteria

- [ ] **AC1** — Página `/empresa/config` com formulário de personalização da empresa
- [ ] **AC2** — Campos editáveis: nome da empresa, tema visual (select com 4 opções), ambient music (toggle), idle timeout lounge (slider 1-30 min), idle timeout break (slider 5-60 min)
- [ ] **AC3** — Preview do tema em tempo real: ao mudar o tema no select, o escritório Phaser atrás do formulário muda imediatamente (sem reload)
- [ ] **AC4** — API endpoint `PUT /api/company-config` para guardar alterações (validação de campos obrigatórios)
- [ ] **AC5** — Feedback visual ao salvar: toast de sucesso ou erro
- [ ] **AC6** — O formulário carrega valores actuais da `company_config` via `GET /api/company-config`
- [ ] **AC7** — Broadcast WS `theme:change` quando o tema é alterado, para que outros clientes abertos actualizem
- [ ] **AC8** — Link na Navbar para `/empresa/config` (ícone de engrenagem junto ao modo Empresa)
- [ ] **AC9** — Design consistente com o resto da UI (Tailwind, dark mode, monospace)
- [ ] **AC10** — `npm run typecheck` e `npm run build` passam sem erros

- [x] **Task 1** — Criar `PUT /api/company-config` endpoint (AC: 4, 7)
  ```typescript
  // src/app/api/company-config/route.ts (adicionar ao ficheiro existente da 4.4)
  import { NextResponse, type NextRequest } from 'next/server';
  import { getCompanyConfig, updateCompanyConfig } from '@/lib/queries';
  import { broadcast } from '@/server/ws-broadcaster';
  import type { ThemeName } from '@/lib/types';

  const VALID_THEMES: ThemeName[] = ['espacial', 'moderno', 'oldschool', 'cyberpunk'];

  export async function PUT(req: NextRequest) {
    try {
      const body = await req.json();

      // Validação
      if (body.theme && !VALID_THEMES.includes(body.theme)) {
        return NextResponse.json({ error: 'Invalid theme' }, { status: 400 });
      }
      if (body.idle_timeout_lounge !== undefined) {
        const val = Number(body.idle_timeout_lounge);
        if (val < 60 || val > 1800) {
          return NextResponse.json({ error: 'idle_timeout_lounge must be 60-1800' }, { status: 400 });
        }
      }
      if (body.idle_timeout_break !== undefined) {
        const val = Number(body.idle_timeout_break);
        if (val < 300 || val > 3600) {
          return NextResponse.json({ error: 'idle_timeout_break must be 300-3600' }, { status: 400 });
        }
      }

      updateCompanyConfig(body);
      const updated = getCompanyConfig();

      // Broadcast theme change se tema mudou
      if (body.theme) {
        try { broadcast({ type: 'theme:change', theme: body.theme }); } catch { /* fire-and-forget */ }
      }

      return NextResponse.json(updated);
    } catch {
      return NextResponse.json({ error: 'Failed to update config' }, { status: 500 });
    }
  }
  ```

- [x] **Task 2** — Adicionar `updateCompanyConfig()` em `queries.ts` (AC: 4)
  ```typescript
  interface CompanyConfigUpdate {
    name?: string;
    theme?: ThemeName;
    ambient_music?: 0 | 1;
    idle_timeout_lounge?: number;
    idle_timeout_break?: number;
  }

  export function updateCompanyConfig(updates: CompanyConfigUpdate): void {
    const fields: string[] = [];
    const values: (string | number)[] = [];

    if (updates.name !== undefined) { fields.push('name = ?'); values.push(updates.name); }
    if (updates.theme !== undefined) { fields.push('theme = ?'); values.push(updates.theme); }
    if (updates.ambient_music !== undefined) { fields.push('ambient_music = ?'); values.push(updates.ambient_music); }
    if (updates.idle_timeout_lounge !== undefined) { fields.push('idle_timeout_lounge = ?'); values.push(updates.idle_timeout_lounge); }
    if (updates.idle_timeout_break !== undefined) { fields.push('idle_timeout_break = ?'); values.push(updates.idle_timeout_break); }

    if (fields.length === 0) return;

    fields.push("updated_at = datetime('now')");
    const sql = `UPDATE company_config SET ${fields.join(', ')} WHERE id = 1`;
    db.prepare(sql).run(...values);
  }
  ```

  'use client';

  import { useState, useEffect } from 'react';
  import type { CompanyConfig, ThemeName } from '@/lib/types';

  const THEMES: { value: ThemeName; label: string; description: string }[] = [
    { value: 'moderno', label: 'Moderno', description: 'Clean, minimal, cores neutras' },
    { value: 'espacial', label: 'Espacial', description: 'Azul profundo, estrelas, glow cyan' },
    { value: 'oldschool', label: 'Oldschool', description: 'Madeira, tons quentes, telas CRT verdes' },
    { value: 'cyberpunk', label: 'Cyberpunk', description: 'Neon, rosa/cyan, scanlines' },
  ];

  export default function CompanyConfigPage() {
    const [config, setConfig] = useState<CompanyConfig | null>(null);
    const [saving, setSaving] = useState(false);
    const [toast, setToast] = useState<{ type: 'success' | 'error'; message: string } | null>(null);

    useEffect(() => {
      fetch('/api/company-config')
        .then((r) => r.json())
        .then(setConfig)
        .catch(() => setToast({ type: 'error', message: 'Erro ao carregar configuração' }));
    }, []);

    async function handleSave() {
      if (!config) return;
      setSaving(true);
      try {
        const res = await fetch('/api/company-config', {
          method: 'PUT',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({
            name: config.name,
            theme: config.theme,
            ambient_music: config.ambient_music,
            idle_timeout_lounge: config.idle_timeout_lounge,
            idle_timeout_break: config.idle_timeout_break,
          }),
        });
        if (!res.ok) throw new Error();
        setToast({ type: 'success', message: 'Configuração guardada!' });
      } catch {
        setToast({ type: 'error', message: 'Erro ao guardar' });
      } finally {
        setSaving(false);
        setTimeout(() => setToast(null), 3000);
      }
    }

    if (!config) return <div className="p-8 text-gray-500 font-mono">Carregando...</div>;

    return (
      <div className="max-w-2xl mx-auto p-8 space-y-8">
        <h1 className="text-2xl font-bold font-mono text-gray-100">
          Configuração da Empresa
        </h1>

        {/* Nome */}
        <div className="space-y-2">
          <label className="block text-sm font-mono text-gray-400">Nome da Empresa</label>
          <input
            type="text"
            value={config.name}
            onChange={(e) => setConfig({ ...config, name: e.target.value })}
            className="w-full bg-gray-800 border border-gray-700 rounded px-3 py-2
                       text-gray-100 font-mono focus:border-blue-500 focus:outline-none"
          />
        </div>

        {/* Tema */}
        <div className="space-y-2">
          <label className="block text-sm font-mono text-gray-400">Tema Visual</label>
          <div className="grid grid-cols-2 gap-3">
            {THEMES.map((t) => (
              <button
                key={t.value}
                onClick={() => setConfig({ ...config, theme: t.value })}
                className={`p-3 rounded border font-mono text-left transition
                  ${config.theme === t.value
                    ? 'border-blue-500 bg-blue-500/10 text-blue-300'
                    : 'border-gray-700 bg-gray-800 text-gray-400 hover:border-gray-600'
                  }`}
              >
                <div className="font-bold">{t.label}</div>
                <div className="text-xs mt-1 opacity-70">{t.description}</div>
              </button>
            ))}
          </div>
        </div>

        {/* Idle Timeouts */}
        <div className="space-y-4">
          <label className="block text-sm font-mono text-gray-400">Timeouts de Inactividade</label>

          <div className="space-y-1">
            <div className="flex justify-between text-xs font-mono text-gray-500">
              <span>Working → Idle (lounge)</span>
              <span>{Math.round(config.idle_timeout_lounge / 60)} min</span>
            </div>
            <input
              type="range" min={60} max={1800} step={60}
              value={config.idle_timeout_lounge}
              onChange={(e) => setConfig({ ...config, idle_timeout_lounge: Number(e.target.value) })}
              className="w-full accent-blue-500"
            />
          </div>

          <div className="space-y-1">
            <div className="flex justify-between text-xs font-mono text-gray-500">
              <span>Idle → Break (café)</span>
              <span>{Math.round(config.idle_timeout_break / 60)} min</span>
            </div>
            <input
              type="range" min={300} max={3600} step={60}
              value={config.idle_timeout_break}
              onChange={(e) => setConfig({ ...config, idle_timeout_break: Number(e.target.value) })}
              className="w-full accent-blue-500"
            />
          </div>
        </div>

        {/* Ambient Music */}
        <div className="flex items-center gap-3">
          <label className="text-sm font-mono text-gray-400">Música Ambiente</label>

        </div>

          onClick={handleSave}

        >
          {saving ? 'Guardando...' : 'Guardar Configuração'}

## Dev Notes

### Fluxo de personalização

```
/empresa/config (formulário)
    → PUT /api/company-config → DB update
    → broadcast WS { type: 'theme:change', theme }
    → PhaserGame.tsx → useEffect detecta theme:change
    → bridge.setTheme(theme) → OfficeScene.setTheme()
    → Redraw floor, walls, furniture, ambient effects
```

### Preview vs Save

O tema muda em tempo real ao clicar no card de tema (preview). O "Guardar" persiste todas as alterações no DB. Se o utilizador mudar de tema mas não guardar, o preview reverte ao recarregar a página.

**Nota**: O preview envia PUT parcial para trigger o WS broadcast. Isto é uma simplificação — numa versão mais robusta, o preview seria local (via bridge directo) e o save seria o único a persistir.

### Validação de timeouts

- `idle_timeout_lounge`: 60s (1min) a 1800s (30min) — working → idle
- `idle_timeout_break`: 300s (5min) a 3600s (60min) — idle → break
- O timeout offline (break → offline) é fixo em 3600s (1h), não configurável

### WS message type

O `theme:change` é um novo tipo de mensagem WS. Adicionado ao `WsIncomingMessage` union type. O `useWebSocket` hook já aceita qualquer mensagem via `lastMessage`, por isso funciona sem alteração.
