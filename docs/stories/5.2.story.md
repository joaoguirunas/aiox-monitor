# Story 5.2 — npm run setup — Comando Único de Instalação

## Story

**Como** utilizador que acaba de clonar o aiox-monitor,
**Quero** um único comando `npm run setup` que instale tudo (deps, DB, hook, PM2),
**Para que** a instalação seja trivial e não precise ler documentação extensa.

## Acceptance Criteria

- [ ] **AC1** — Script `scripts/setup.sh` executa toda a instalação sequencialmente
- [ ] **AC2** — O setup instala dependencies (`npm install`)
- [ ] **AC3** — O setup cria o diretório `data/` e inicializa o SQLite DB (schema)
- [ ] **AC4** — O setup instala o hook Python no Claude Code (`npm run install-hook`)
- [ ] **AC5** — O setup copia `.env.example` para `.env` se não existir
- [ ] **AC6** — O setup faz `npm run build` para gerar o build de produção
- [ ] **AC7** — O setup mostra instruções claras no final (como iniciar, URL, PM2 startup)
- [ ] **AC8** — O setup é idempotente: pode ser executado múltiplas vezes sem problemas
- [ ] **AC9** — O setup detecta erros e mostra mensagens claras (ex: Node.js versão mínima)
- [ ] **AC10** — `npm run setup` está definido no `package.json`

## Tasks / Subtasks

- [x] **Task 1** — Criar `scripts/setup.sh` com toda a lógica de instalação (AC: 1-9)
  ```bash
  #!/bin/bash
  set -e

  echo "🚀 aiox-monitor — Setup"
  echo "========================"

  # Check Node.js version (>=18)
  NODE_VERSION=$(node -v | cut -d'v' -f2 | cut -d'.' -f1)
  if [ "$NODE_VERSION" -lt 18 ]; then
    echo "❌ Node.js 18+ required (found: $(node -v))"
    exit 1
  fi

  # Install dependencies
  echo "📦 Installing dependencies..."
  npm install

  # Create data directory
  echo "📁 Creating data directory..."
  mkdir -p data logs

  # Copy .env if needed
  if [ ! -f .env ]; then
    echo "📝 Creating .env from .env.example..."
    cp .env.example .env
  fi

  # Build
  echo "🔨 Building..."
  npm run build

  # Init DB (start server briefly to trigger schema creation)
  echo "🗄️ Initializing database..."
  timeout 5 npx tsx server.ts || true

  # Install hook
  echo "🪝 Installing Claude Code hook..."
  npm run install-hook || echo "⚠️  Hook install skipped (Claude Code not detected)"

  echo ""
  echo "✅ Setup complete!"
  echo ""
  echo "  Start:    npm run pm2:start"
  echo "  Dev:      npm run dev"
  echo "  URL:      http://localhost:8888"
  echo "  Startup:  npm run pm2:startup (auto-start on boot)"
  ```

- [x] **Task 2** — Adicionar `npm run setup` ao `package.json` (AC: 10)
  ```json
  {
    "setup": "bash scripts/setup.sh"
  }
  ```

- [x] **Task 3** — Validar `.env.example` existe e tem os campos necessários (AC: 5)

- [x] **Task 4** — Validar typecheck e build

## Dev Notes

- O script usa `set -e` para parar em qualquer erro
- `timeout 5` inicia o server brevemente só para criar o schema — mata após 5s
- O hook install pode falhar silenciosamente se Claude Code não estiver instalado
- Idempotente: `mkdir -p`, `cp` com check, `npm install` é safe para re-run
