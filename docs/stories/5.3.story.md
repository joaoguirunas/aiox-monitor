# Story 5.3 — README Completo para Open Source

## Story

**Como** developer que descobre o aiox-monitor no GitHub,
**Quero** um README claro, visual e completo que explique o que é, como instalar e como usar,
**Para que** consiga começar a usar em minutos sem dúvidas.

  ## Features

  - 🏢 **Virtual Office** — Isometric Phaser.js office where agents walk, sit, and work
  - 📊 **Kanban Board** — Projects as columns, agents as cards, real-time status
  - 📋 **Event Log** — Filterable table of all agent events
  - 🎨 **4 Themes** — Moderno, Espacial, Oldschool, Cyberpunk
  - ⚡ **Real-time** — WebSocket for instant updates
  - 🔌 **Claude Code Hook** — Auto-detects agents, tools, and sessions

  ## Quick Start

  ```bash
  git clone https://github.com/joaoguirunas/aiox-monitor.git
  cd aiox-monitor
  npm run setup
  npm run dev        # http://localhost:8888
