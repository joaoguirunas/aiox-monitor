# Story 8.1 — Fix Hyphenated Path Conversion in JSONL Watcher

## Story

**Como** sistema de monitorização,
**Quero** que `findTerminalByProjectDir` converta correctamente paths de projecto com hífens no nome,
**Para que** o JSONL enrichment (tool detail, permission detection) funcione para projectos como `aiox-monitor`.

## Context

O Claude Code armazena transcripts JSONL em `~/.claude/projects/<hash>/<session>.jsonl`. O `<hash>` é o path do projecto com `/` substituídos por `-` (ex: `/Users/joaoramos/Desktop/aiox-monitor` → `-Users-joaoramos-Desktop-aiox-monitor`).

**Bug actual:** `jsonl-watcher.ts:75` tenta reverter esta conversão com `replace(/-/g, '/')`, mas isso destrói hífens legítimos no nome do projecto: `-Users-joaoramos-Desktop-aiox-monitor` → `/Users/joaoramos/Desktop/aiox/monitor` (errado). O match com `projects.path` falha, e o JSONL enrichment nunca chega ao terminal correcto.

**Referência:** `docs/Terminais.md` secção 5.1 | Epic 8 Bug Ref 5.1

## Acceptance Criteria

### Task 1: Corrigir conversão de path
- [x] **AC1** — Substituir a lógica em `findTerminalByProjectDir()` (`src/server/jsonl-watcher.ts:72-91`): em vez de converter o directório Claude para path real (`replace(/-/g, '/')`), converter cada `projects.path` para formato Claude (`'/' → '-'`) e comparar directamente
- [x] **AC2** — A conversão de `projects.path` para formato Claude deve ser: prefixar com `-`, substituir `/` por `-` (ex: `/Users/joaoramos/Desktop/aiox-monitor` → `-Users-joaoramos-Desktop-aiox-monitor`)
- [x] **AC3** — Iterar sobre todos os projectos na DB para encontrar match (query `SELECT id, path FROM projects`), comparar `claudeFormat === projectDirName`
- [x] **AC4** — Quando match é encontrado, retornar o terminal mais recente não-inactivo para esse projecto (query existente mantém-se)

- [ ] **AC6** — Projectos sem hífen (ex: `myapp`) continuam a funcionar — a nova lógica é compatível

### Abordagem correcta
```typescript
function findTerminalByProjectDir(projectDirName: string): { id: number; project_id: number } | null {
  try {
    const projects = db.prepare(`SELECT id, path FROM projects`).all() as Record<string, unknown>[];
    for (const proj of projects) {
      const projPath = proj.path as string;
      // Convert project path to Claude's directory format for comparison
      const claudeFormat = '-' + projPath.slice(1).replace(/\//g, '-');
      if (claudeFormat === projectDirName) {
        const row = db.prepare(
          `SELECT id, project_id FROM terminals WHERE project_id = ? AND status != 'inactive' ORDER BY last_active DESC LIMIT 1`
        ).get(proj.id as number) as Record<string, unknown> | undefined;
        if (row) return { id: row.id as number, project_id: row.project_id as number };
        return null;
      }
    }
  } catch { /* ignore */ }
  return null;
}
```
