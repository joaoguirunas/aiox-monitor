# Story 9.6 — Filtro por Período + Duração de Sessão

## Story

**Como** utilizador do aiox-monitor,
**Quero** filtrar sessões e eventos por período temporal e ver a duração de cada sessão,
**Para que** consiga investigar actividade num intervalo específico e entender quanto tempo cada comando demorou.

## Context

A FilterBar não tem filtros temporais — o utilizador vê tudo sem possibilidade de focar num dia ou hora. As sessões têm `started_at` e `ended_at` no banco mas a duração não é calculada nem exibida. Os endpoints já suportam `since` — falta adicionar `until` e expor na UI.

## Dev Notes

### Formatação de duração

```typescript
function formatDuration(startedAt: string, endedAt?: string): string {
  const start = new Date(startedAt).getTime();
  const end = endedAt ? new Date(endedAt).getTime() : Date.now();
  const ms = end - start;
  const mins = Math.floor(ms / 60000);
  const secs = Math.floor((ms % 60000) / 1000);
  if (mins >= 60) return `${Math.floor(mins/60)}h ${mins%60}m`;
  return `${mins}m ${secs}s`;
}
```
